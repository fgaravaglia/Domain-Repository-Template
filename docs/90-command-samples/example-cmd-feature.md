# Command Template: Feature

this is the template to provide a command to implement a new feature

## Command Prompt

Implement [feature name] with the following requirements:

**User Story:** As a user, I want to [action] so that [benefit].

**Acceptance Criteria:**
[ ] Feature is accessible from [location]
[ ] Data persists across sessions
[ ] Loading states are handled
[ ] Errors display user-friendly messages

Check for conflicts with existing code before implementing.
